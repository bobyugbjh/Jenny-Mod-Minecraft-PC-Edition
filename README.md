# 🚀 Auto Installer (Jenny-Mod-Installer.exe)

The included installer simplifies the entire process.

### Steps

1. Run **Jenny-Mod-Installer.exe** as Administrator.
2. The installer will automatically detect your **Minecraft Java Edition** folder.
3. Click **Install**.
4. The tool will:
   - Install Forge if required
   - Copy the Jenny Mod into the `mods` folder
   - Create necessary configuration files
5. Launch Minecraft using the **Forge profile**.

After loading the world, the mod will be active automatically.

> ℹ️ The installer is compiled as a standalone executable to reduce dependency issues. Some antivirus tools may flag it as unknown software.

# 🧩 Manual Installation (Optional)

If you prefer manual setup:

1. Install **Minecraft Forge**.
2. Open your `.minecraft` directory.
3. Navigate to the `mods` folder.
4. Copy `Jenny-Mod.jar` into the folder.
5. Launch the game using the **Forge profile**.

# ❗ Troubleshooting

| Problem | Solution |
|--------|----------|
| Mod not loading | Make sure Forge is installed and selected in the launcher |
| Game crashes on startup | Check Minecraft version compatibility |
| NPC not spawning | Try creating a new world |
| Mods folder missing | Launch Forge once to generate the folder |
| Installer blocked by antivirus | Temporarily allow the file or add an exclusion |
